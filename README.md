        导出：export default {}

路由的生命周期：
    入口main.js中还有两个：beforeEach  beforeResolve
    离开一个组件时：  beforeRouteLeave 
    进入到一个新的页面：beforeEach
    router配置文件中也有路由钩子：beforeEnter
    进入新的组件：beforeRouteEnter 
    解析完成：beforeResolve 
    进入完毕： afterEach
    路径发生改变：beforeRouteUpdate
